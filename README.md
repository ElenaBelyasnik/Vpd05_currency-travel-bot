# Конвертер валют (AllRatesToday API)

CLI-инструмент для получения курсов валют и конвертации сумм через [AllRatesToday API](https://allratestoday.com).

## Возможности

- Получение актуального курса между двумя валютами
- Конвертация суммы из одной валюты в другую
- Поддержка нескольких целевых валют (через запятую)
- Интерактивное CLI-меню или запуск с аргументами
- Цветной вывод (colorama)
- Безопасное хранение API-ключа через `.env`

## Установка

### 1. Клонируйте репозиторий

```bash
git clone <ваш_репозиторий>
cd currency_converter_cli
```

### 2. Создайте виртуальное окружение

```bash
python -m venv venv
```

**Windows:**
```bash
venv\Scripts\activate
```

**Linux/macOS:**
```bash
source venv/bin/activate
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Настройте API-ключ

Создайте файл `.env` в корне проекта:

```
CURRENCY_API_KEY=ваш_ключ_из_allratestoday
```

> Получите ключ на [allratestoday.com](https://allratestoday.com)

## Использование

### Интерактивный режим

Запустите без аргументов:

```bash
python cli.py
```

Вы увидите меню:
```
========================================
  Конвертер валют (AllRatesToday)
========================================
  1. Получить курс валюты
  2. Конвертировать сумму
  3. Показать список валют
  4. Выход
========================================
```

### Режим CLI-аргументов

**Получить курс:**
```bash
python cli.py --rate USD EUR
```

**Конвертировать сумму:**
```bash
python cli.py --convert 1000 USD EUR
```

**Несколько целевых валют:**
```bash
python cli.py --rate USD EUR,GBP,JPY
```

## Безопасность

- API-ключ хранится в `.env` и не попадает в git
- Виртуальное окружение `venv/` игнорируется
- Папка `drafts/` игнорируется

## Обработка ошибок

Программа корректно обрабатывает:
- Неверный API-ключ (401)
- Превышение лимита запросов (429)
- Ошибки сервера (500+)
- Отсутствие интернета
- Неверные коды валют (404)

## Требования

- Python 3.8+
- `requests`
- `python-dotenv`
- `colorama` (опционально, для цветного вывода)

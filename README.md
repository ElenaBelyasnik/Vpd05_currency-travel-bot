# 💰 Финансовый помощник для путешествий

Python-приложение (Telegram-бот + CLI) для управления бюджетом в путешествиях с автоматической конвертацией валют.

## ✨ Возможности

- 🌍 Создание путешествий между странами
- 💱 Автоматическая конвертация валют через API ЦБ РФ (ежедневные курсы)
- 📊 Отслеживание баланса в двух валютах одновременно
- 📜 История расходов с конвертацией
- 🔄 Переключение между активными путешествиями
- 📈 Изменение курса вручную
- 🗑 Очистка данных

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка Telegram-бота

1. Создай бота через [@BotFather](https://t.me/BotFather)
2. Скопируй `.env.example` в `.env` и вставь токен:
   ```bash
   cp .env.example .env
   ```
   или создай `.env` вручную:
   ```
   TELEGRAM_BOT_TOKEN=your_token_here
   ```

### 3. Запуск

**Telegram-бот:**
```bash
python frontend/telegram_bot.py
```

**CLI-интерфейс (для тестирования):**
```bash
python frontend/cli_interface.py
```

## 📱 Как пользоваться ботом

### Создание путешествия
1. `/start` — главное меню
2. "➕ Создать путешествие"
3. Выбери страну отправления и назначения
4. Введи бюджет в валюте отправления
5. Бот рассчитает курс и покажет итоговую сумму

### Учёт расходов
- Просто отправь число — бот учтёт его как расход
- Или используй меню "💰 Баланс" и "📜 История расходов"

### Управление путешествиями
- "📋 Мои путешествия" — список всех поездок
- "📊 Изменить курс" — обновить курс вручную
- "🗑 Очистить данные" — удалить все данные (осторожно!)

## 🌍 Поддерживаемые валюты

25 валют, включая: RUB, USD, EUR, GBP, JPY, CNY, CHF, CAD, AUD, INR, TRY и другие.

Поддерживается 50+ стран, включая все государства Евросоюза (EUR), США, Великобританию, Японию, Китай, Турцию и многие другие.

## 💱 Как работает конвертация

Бот использует бесплатный API ЦБ РФ: `https://www.cbr-xml-daily.ru/daily_json.js`

Курсы обновляются ежедневно. При создании путешествия фиксируется текущий курс.

## 📊 База данных

Проект использует SQLite3. Файл создаётся автоматически в `database/travel_bot.db`.

**Таблицы:**
- `users` — пользователи бота
- `trips` — путешествия (is_active = 1 означает активное)
- `expenses` — расходы по путешествиям

## ⚙️ Технические детали

- **Python 3.7+**
- **pyTelegramBotAPI** — библиотека для Telegram-бота
- **SQLite3** — встроенная база данных
- **requests** — HTTP-клиент для API ЦБ РФ
- **colorama** — цветной вывод в CLI (Windows + Linux/Mac)

# Конвертер валют ЦБ РФ

Python-модуль для работы с курсами валют через API Центрального Банка РФ. Использует бесплатную JSON-обёртку: https://www.cbr-xml-daily.ru/daily_json.js

## Структура проекта

```
├── convert_currency/
│   └── currency_util.py    # Основной модуль
├── tests/
│   └── test_currency_cli.py # CLI-интерфейс для тестирования
└── README.md
```

## Требования

- Python 3.7+
- Библиотека `requests`

Установка зависимостей:
```bash
pip install requests
```

## Использование

### Импорт модуля

```python
from convert_currency.currency_util import (
    get_currency_code_by_country,
    get_currency_pair_rate,
    CURRENCIES,
    COUNTRY_TO_CURRENCY,
)

# Получить код валюты по названию страны
code = get_currency_code_by_country('Россия')  # 'RUB'
code = get_currency_code_by_country('Германия')  # 'EUR'

# Получить курс валют
rate = get_currency_pair_rate('USD', 'RUB')  # курс USD к RUB
rate = get_currency_pair_rate('EUR', 'JPY')  # курс EUR к JPY
```

### Командная строка

**Получить курс валют двух стран:**
```bash
python convert_currency/currency_util.py <Страна1> <Страна2>
```

Примеры:
```bash
python convert_currency/currency_util.py Россия США
python convert_currency/currency_util.py Германия Япония
```

**Получить конвертированную сумму:**
```bash
python convert_currency/currency_util.py <Страна1> <Страна2> <сумма>
```

Примеры:
```bash
python convert_currency/currency_util.py Россия США 10000
python convert_currency/currency_util.py Германия Япония 1000
```

### CLI-интерфейс

Запуск интерактивного меню:
```bash
python tests/test_currency_cli.py
```

Меню предлагает:
1. Список всех доступных валют и стран
2. Конвертацию между валютами двух стран с вводом суммы
3. Выход

## Обработка ошибок

Модуль обрабатывает следующие ошибки:
- `CurrencyAPIError` — ошибка подключения к API, таймаут, HTTP-ошибка, неверный код валюты
- `ValueError` — некорректное числовое значение суммы
- Страна не найдена в словаре
